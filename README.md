# Astral Life

WIP game

## Build

### Linux / Mac

For linux, install the dependencies:

 * glfw
 * freetype
 * openal-soft

To build:

```
./build.sh
```

Optionally pass `--debug` or `--release`.

### Windows

Install MSBuild Tools, open the VS Developer Console. `cd` to the repo, then:

```
.\build.bat
```

Optionally pass `--debug` or `--release`.

## Run

### Linux / Mac

```
./build/astral-life
```

### Windows

Release (default):

```
.\build\Release\astral-life.exe
```

Debug:

```
.\build\Debug\astral-life.exe
```
