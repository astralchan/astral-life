# Astral Life

WIP game

## Dependencies

 * glfw
 * vulkan-headers
 * openal
 * freetype

## Build

### Windows

Open the VS Developer Console, cd to the directory and run:

```
.\build.bat
```

For the debug build, run:

```
.\build.bat --debug
```

### Linux / Mac

```
./build.sh
```

For the debug version:

```
./build.sh --debug
```

## Run

### Windows

For release build:

```
.\build\Release\astral-life.exe
```

For debug build:

```
.\build\Debug\astral-life.exe
```

### Linux / Mac

```
./build/astral-life
```
